<p>The nRF Toolbox is a container app that stores your Nordic Semiconductor apps for Bluetooth Smart in one location.  </p>

<h2>

<p>The Device Firmware Update (DFU) profile allows you to update the application, bootloader and/or the Soft Device image over-the-air (OTA). It is compatible with Nordic Semiconductor nRF51822, nRF51422 devices that have the S110 SoftDevice and bootloader enabled. </p>

<ul class="task-list">
<li>
<strong>Scans for devices that have DFU service</strong>, </li>
<li>
<strong>Uploads the selected firmware in Zip format. See nRF-Utility</strong>. </li>
<li>
<strong>Includes pre-installed examples that consist of a blinky firmware.</strong> </li>
<li>
<strong>Updates status</strong> </li>
</ul>

"browse" button - Choose a firmware to update. The zip file should be packed by using nRF-Utility.

<h3>
<a id="user-content-upload-limitation" class="anchor" href="#upload-limitation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Upload limitation</h3>

<p>The application sends 20-bytes size packets at a time.</p>
